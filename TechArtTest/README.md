# Dashy Crashy

## Overview
Dashy Crashy is a mobile endless runner developed in Unity where the player controls a car and avoids obstacles to survive as long as possible.

## Approach
- Implemented lane-based movement
- Infinite road illusion using looping environment
- Obstacle spawning system
- Score system based on survival time
- UI designed for clarity on mobile
- Iterative development (core → polish)

## Assets & Tools Used
Engine:
- Unity (Mobile)

Tools:
- Adobe Photoshop (UI, textures)
- Blender (3D models)

Assets:
- Mix of self-created and store/free assets

## Optimization Decisions
- Reduced draw calls using shared materials
- Used low poly models
- Compressed textures for mobile
- Basic object reuse (pooling concept)
- Limited active objects
- Optimized UI scaling

## Key Focus
- Responsive controls
- Endless gameplay loop
- Mobile performance

## Future Improvements
- Advanced pooling
- Difficulty scaling
- Better VFX
